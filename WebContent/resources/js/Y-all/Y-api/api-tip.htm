构造函数

构造函数 描述
Tip 在目标对象附近生成一个提示

配置参数

配置参数 参数说明 描述
target jquery选择器或对象,必须。tip相关对象 无
autoDisappear true/false,默认true。 是否自动消失（指点击组件外任意位置）
align 'top'/'bottom'/'left'/'top' 或其组合如 'left top',默认'right bottom'。 相对target的位置
spacing 数值，默认0。 在相对位置基础上的偏移
其他 无 参见window配置

方法 参数说明 返回值 描述
show(speed) 参见Y.Window 显示组件
close(speed) 参见Y.box 关闭组件
其他 参见Y.Window
