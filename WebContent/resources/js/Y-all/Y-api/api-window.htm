构造函数

构造函数 描述
Window 生成一个弹出层，根据参数可有模态，非模态；
非模态下多个窗体共存时支持点击前置

配置参数

配置参数 参数说明 描述
renderTo jquery选择器或对象,或html字符串,默认'body' 渲染位置，一般都应该不配置或配置为'body'
title 字符串 标题
content jquery选择器或对象,或html字符串，必须 组件内容
contentClone true/false,默认false 为true则渲染克隆内容而非原内容
contentRecovery true/false,默认true 为true时，且渲染内容为原页面内容（且未clone）时，销毁时将内容渲染到原容器
removeLocation true/false,默认true 为true移除content的样式中的position，left，top 属性
dragAble true/false,默认true 是否激活拖拽
autoSize true/false,默认true 为true时会根据内容尺寸自动修正大小(若内容尺寸不确定,请不要使用该功能)
modal true/false,默认true 是否启用遮罩
noheader true/false,默认false 若该项为true,则组件将不渲染标题栏,同时默认不激活拖拽,
若仍然需要激活拖拽,需要配置dragEle项为组件指定一个触发拖拽的元素,
同理若还需要关闭,可配置closeEle
noStyle true/false,默认false 为true则无样式渲染组件
simple true/false,默认false 为true时以最简单模式渲染组件，content内容即为组件UI
因此有关样式的配置将通通失效(组件位置的配置仍然有效)，也没有header,body的结构，
同时失效的还有noheader(必然为true)，contentClone,title,closeBtn,autoSize,width,height,
el,wndEle,headerStyle,bodyStyle等；
dragEle jquery选择器字符串 在组件find一个当没有标题栏时触发拖拽的元素
closeEle jquery选择器字符串 在组件find一个当没有标题栏时触发关闭的元素
closeAction 可选值为'hide'/'close' 为hide时关闭组件时调用hide方法而非distroy方法
noManager true/false，默认false 为true时不设置管理器，不应配置该项，
无管理器无法确保弹出层显示在最上层
el 字符串，类似 'div'或 'span' 指定组件最外层容器标签类型
width 数值，默认无 配置可自动组件宽度
height 数值，默认无 配置可自动组件高度
left 数值，默认无 配置可自动组件left位置
top 数值，默认无 配置可自动组件top位置
wndStyle jquery的css方法支持的样式对象 作用于组件最外层容器的样式
headerStyle jquery的css方法支持的样式对象 作用于组件header的样式
bodyStyle jquery的css方法支持的样式对象 作用于组件body的样式
overStyle jquery的css方法支持的样式对象 作用于组件遮罩层的样式
show 事件函数 组件显示时调用
close 事件函数 组件关闭时调用
beforeshow 事件函数 组件显示前，返回false终止显示
beforeclose 事件函数 组件关闭前调用，返回false终止关闭
distroy 事件函数 组件销毁时调用
其他 无 参见Y.box配置

对象方法

方法 参数说明 返回值 描述
show(speed) speed为数字，显示速度，可不配置，默认为'fast' false或无（被beforeshow阻止时返回false） 显示组件
close(speed) speed为数字，隐藏速度，可不配置，默认为'fast' false或无 关闭组件
hide(speed) speed为数字，隐藏速度，可不配置，默认为'fast' 无 隐藏组件
distroy() 无 销毁组件，清除注册在dom上的事件,
默认情况下close方法会调用此方法,除非配置closeAction为'hide'
其他 参见Y.box方法

对象属性

属性 描述
el 最外层容器jquery对象
wnd 同el，最外层容器jquery对象
body 主体部分jquery对象
item 内容部分jquery对象，即content配置
manager 管理器，管理器负责管理多个window组件的显示zindex，
一般情况下不需要对该对象进行操作
其他 参见Y.box
