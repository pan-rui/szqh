配置参数

配置参数 参数说明 描述
eleArr 必须:所有需要展示的img对象 请传入JQ选择器字符串 为了配合live 让新加的满足条件的img也能有此功能
renderTo 默认:'body' 组件渲染的位置 一般不需要配置
wheelSts 默认:true 鼠标中键放大缩小功能
moveCutSts 默认:true 左右分屏点击 实现前后跳转功能
showAfter 默认:null 此参数为函数 如果是字符串则为内部方法 显示图片后调用方法 方法第一个参数为img 第二个参数为整个外层DIV
pathInfo string:显示标签的属性,function:执行函数后的返回值为显示的信息 函数内 this是要显示的img对象 信息会显示到第一个p 无pathEle 则不需要配置
titleInfo string:显示标签的属性,function:执行函数后的返回值为显示的信息 函数内 this是要显示的img对象 信息会显示到第二个p :无titleEle 则不需要配置
minScale 默认:1 最小缩小比例
maxScale 默认:2 最大放大比例
mainDiv 默认:null null时会生成默认模板 总的页面对象 如果不配置 下面的参数都可以不用配置
imgDiv 默认:'.main-photo' 放置图片的div
cwBtn 默认:'.rota-r' 右旋按钮
acwBtn 默认:'.rota-l' 左旋按钮
biggerBtn 默认:'.magnify' 放大按钮
smallerBtn 默认:'.lessen' 缩小按钮
fullBtn 默认:'.full' 全屏按钮
prevBtn 默认:'.prev-photo' 上一张按钮
nextBtn 默认:'.next-photo' 下一张按钮
titleEle 默认:'.info-title' 第一行信息
pathEle 默认:'.path-title' 第二行信息
其他 无 参见window配置

对象方法

方法 参数说明 返回值 描述
transformHandler(num) 旋转的角度 90的倍数 可以是0 负数 返回组件本身 使图片旋转一定角度
zoomHandler(num) 放大缩小 增加的倍率 0.1 -0.1 返回组件本身 使图片放大缩小一定比例
toFull() 无 返回组件本身 全屏
cancelFull() 无 返回组件本身 取消全屏
setEle(arg) 可以是字符串'+1','-1' 是向前向后X张;可以是数字1 跳转到X张;可以是img对象 显示这个img 返回组件本身 设置要显示的图片
toMoveCenter() 不需要参数 返回组件本身 将图片居中 一般情况不需要调用此方法
getZoom() 不需要参数 Number 返回当前缩放比例
getTransform() 不需要参数 Number 返回当前旋转角度
setZoomScale(a,b) a:minScale; b:maxScale 返回组件本身 设置允许的最小最大缩放比例
