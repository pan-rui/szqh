配置参数

配置参数 参数说明 描述
target jquery选择器或对象,必须。tip相关对象 无
content 要显示的内容'<div>XX</div>' 显示的标签类型
spacing 数字 默认5 生成div与对象的距离
data 序列数组:['a','b'] 提供的搜索对象
showNumbers 数字 默认5 最多显示的结果条数
fontColor 颜色 默认"#f00" 匹配的字符串变色
clickAfter 函数 (ele,val) 第一个参数是点击的item 第二个参数是值 点击item后的回调函数
bindType 默认bind 可选live 绑定事件方式
其他 无 参见Y.Tip配置
