构造函数

构造函数 描述
SelectBranch 省市支行联动

配置参数

配置参数 参数说明 描述
renderTo 默认 #selectDiv 放置内容的对象
bankSelect 默认 #selectBank 银行选择对象
branchUrl url 获取支行信息
provinceCityUrl url 获取省市信息
comboOpts 默认 { province:{name:'province'}, city:{name:'city'}, branch:{name:'branch'} } 设置name属性
其他 无 参见Y.Combox配置

对象方法

方法 参数说明 返回值 描述
setAreaInfo(areaInfo, first, noInit) areaInfo:省市信息,first:初始化信息,noInit:是否需要初始化 返回组件本身 设置省市信息
setValue(info,type) info:设置初始化信息,type:设置属性 默认value 返回组件本身 设置显示的值
getText() 无参数 返回text对象 {province: provinceText, city:cityText, branch:branchText}; 返回对象的text
getValue() 无参数 返回text对象 {province: provinceValue, city:cityValue, branch:branchValue}; 返回对象的Value
handleProvinceList(callback) 回调函数 无 重新初始化对象
toProvinceItems(res) 自定义 [{value:xx,text:xx}] 设置省信息数组
toCityItems(res) 自定义 [{value:xx,text:xx}] 设置城市信息数组
toBranchItems(res) 自定义 [{value:xx,text:xx}] 设置支行信息数组
其他 参见Y.Combox
