构造函数

构造函数 描述
Countdown 在页面显示一个消息框

配置参数

配置参数 参数说明 描述
message 字符串，格式如 xxx{0}xxx，{0}将被替换为时间数字 组件显示的文本消息
time 数字 倒计时时常，默认60，单位秒
space 数字 每次跳动的时间间隔
autoDisappear true/false，默认true 倒计时结束后是否自动关闭组件
timeover 事件函数 计时跳到0时执行该函数
其他 无 参见Y.box配置

对象方法

方法 参数说明 返回值 描述
show(speed) 参见Y.box 显示组件
close(speed) 参见Y.box 关闭组件
setTime(number) 数字 设置实时时间数字
startTimer() 启动计时，如正在计时则重新开始
setMessage(msg) 字符串，格式同message配置 设置显示信息
其他 参见Y.box
