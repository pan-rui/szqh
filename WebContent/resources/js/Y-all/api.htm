概述

Y库为易极付公司自主开发的组件及函数库，Y库基于jQuery，并使用seajs加载

组合框Y.Combobox
弹出层Y.Window
倒计时Y.Countdown

Y库核心

Y库核心成员为Y库的基础
成员 功能描述 作用
all 对象 组件全局印象 所有Y库组件构造时都将加入到全局印象中，以便在需要时获取
getKey 获取一个与已有组件绝不重复的key key成员时Y库组件在全局中的标识，当没有为组件配置key时，组件会自动调用该函数生成key
getCmp 参数 key ，根据key获取组件 当因为作用域或其他原因无法直接获得组件对象时，可为组件配置key， 然后通过key从全局中获得组件对象
join 将一个组件加入到全局印象中 Y库组件构造时会自动调用该函数加入到全局印象中
remove 将一个组件从全局印象中移除 Y库组件销毁时会自动调用该函数
create 创建一个Y库组件function(name,cfg) name为组件名，cfg为组件配置 应该使用该函数创建组件而不应直接new Y.xxx()，可减少异常发生的可能
inherit 为Y库扩展一个成员，并可以继承一个Y库中的基类，function(name,base,option)
name为新组件名，base为基类名，option为新增或被重写的成员		 该函数为Y库的组件扩展方式，为Y库扩展新的组件，并可从已有组件中继承UI及成员
errorList 错误信息数组 存储一些使用中的错误信息
handlerError 异常处理函数 默认将异常信息存储到errorList中

组件调用方式

推荐两种组件创建方式：

1,Y库方式：
Y.create('Countdown',{
     renderTo: '#span1',
     time:60
});
2,Jquery方式：
$('#span1').Y('Countdown',{
     time:60
});
非组件的成员，请根据API中各自的描述使用

基类component（不应构造该类）

配置参数：无
成员 描述 作用
init 空 组件构造函数中会默认调用init方法
callBase 调用基类方法，function(name,base,[param1,param2]);
name为方法名，base为基类名，方法参数在后面顺序书写		 派生类中调用基类成员的方法
on on(name,handler)为组件绑定一个事件函数，参数为事件名和处理函数 Y库组件事件绑定的规范函数
fire (name,[param1],[param2])触发一个事件，第一个从参数为事件名，后面的为传递的参数 将触发事件名配置的事件函数、listeners配置的事件函数、on方法绑定的事件函数
method function(name,[param1,param2]) 根据方法名调用对应方法
distroy 销毁组件

基类box

供有UI的组件继承，提供组件渲染、显示、关闭、销毁等基本流程及功能

配置参数

配置参数 参数说明 描述
renderTo jquery对象或选择器 组件渲染位置
el 字符串，如'div' 组件最外层容器标签名
css class名 加到组件el上的class样式
style 样式对象，如：{background:'xxx',width:'xxx'} 加到组件el上的style样式

对象方法

方法 参数说明 返回值 描述
show(speed) speed为数字，显示速度，可不配置，默认为'fast' false或无（被beforeshow阻止时返回false） 显示组件
close(speed) speed为数字，隐藏速度，可不配置，默认为'fast' false或无 关闭组件
hide(speed) speed为数字，隐藏速度，可不配置，默认为'fast' 无 隐藏组件
distroy() 无 销毁组件，清除注册在dom上的事件,
默认情况下close方法会调用此方法,除非配置closeAction为'hide'

对象属性

属性 描述
el 最外层容器jquery对象
isRender 组件是否已经渲染
isShow 组件是否已经显示
isDistroy 组件是否已经销毁
showwing 组件是否处于显示状态
